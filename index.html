🌳 수목형 (전통적) 모델

  • 지식 구조: 선형적, 위계적, 순차적
  • 시작점: 미리 정해진 시작점 (1장)

  • 실패: 피해야 할 오류

🕸️ 리좀적 (탐구 기반) 모델

  • 지식 구조: 네트워크형, 비선형적
  • 시작점: 모든 지점의 호기심, 질문
  • 학습자 역할: 능동적 탐구자, 지식 창조자
  • 과정: 생성적 경로 탐색, 연결 만들기
  • 목표: 심층적 이해, 문제 해결
  • 실패: 학습 과정의 필수 요소

학습 모델 종합 분석

두 학습 모델의 특징을 다각도에서 시각적으로 비교합니다.

성공적인 탐험을 위한 전략

자유로운 학습의 바다를 항해할 때 마주할 수 있는 어려움과 해결책입니다.

🎯 도전 1: 방향성 상실

문제점: 정보의 바다에서 길을 잃고 압도당함.

해결책: 단기적이고 구체적인 '탐구 목표'를 설정하여 집중점을 만듭니다. (예: "오늘은 이 개념 하나만 파악한다")

🧩 도전 2: 지식 격차

문제점: 흥미 위주로만 탐색하여 필수 기초를 놓침.

해결책: 주기적으로 '전략적 통합' 단계를 갖고, 부족한 기초 개념을 의도적으로 보충합니다.

📉 도전 3: 동기 저하

문제점: 초기의 열정이 식거나 어려운 개념에 부딪힘.

해결책: 마인드맵이나 학습 일지로 과정을 시각화하여 '성장하고 있다'는 유능감을 느끼고 성취감을 얻습니다.
